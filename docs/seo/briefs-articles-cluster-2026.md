# Briefs articles — cluster SEO 2026

Briefs prêts à rédiger pour construire l'autorité thématique d'OpenLab Consulting (cabinet ivoirien d'IA appliquée, R&D produit, édition).

**Règles communes à tous les articles**

- Ton : phrases courtes, directes, adresse au « vous ». Bannir « solutions innovantes », « synergies », « ADN », anglicismes évitables (cf. CLAUDE.md §18).
- Chiffres : aucun chiffre inventé. Tout chiffre est marqué « à sourcer : … ». Pas de chiffre rond non sourcé en ligne.
- Structure : H1 unique, TOC sticky desktop, un H2 toutes les 300-400 mots, 3-5 liens internes, 1-2 sources d'autorité externes, conclusion + CTA.
- SEO : title ≤ 60 caractères avec mot-clé en première moitié, meta description ≤ 155 caractères, résumé en 3 bullets en tête (GEO §12.4), Schema.org `Article` + `BreadcrumbList`.
- Catégories autorisées : `souverainete`, `conformite-rh`, `cybersecurite`, `data-gouvernance`, `agents-ia`, `mlops`, `etude-de-cas`.
